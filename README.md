# Tasty & Full

**[Live Demo](https://eat-real.github.io/tasty-full/)** · **[PRD](./prd.md)**

Weekly meal planning & ordering app. Pick breakfast, lunch, and dinner for each day — see calories & macros, track delivery blocks, and checkout.

## Stack

- Vite + React 18 (SPA, static build)
- Supabase (planned backend — auth, DB, storage)
- GitHub Pages (hosting via GitHub Actions)

## Dev

```bash
npm install
npm run dev
```

## Features

- Weekly meal planner with drag-and-drop
- KBJU (calories, protein, fat, carbs) tracking per day
- Dietary filters (vegan, gluten-free, lactose-free)
- 3 delivery windows/week (Mon, Wed, Fri)
- Free delivery over €50
- 6 languages (EN, PT, ES, FR, UK, RU)
- Responsive mobile + desktop
